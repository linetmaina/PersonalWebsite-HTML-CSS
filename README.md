# Personal Website 🌐

## Overview
This is my personal portfolio website built using HTML and CSS.  
It showcases my skills and projects as a junior software developer.

---

## Features
- Responsive design (works on desktop, tablet, and mobile)
- About Me section
- Skills section
- Projects section displayed using CSS Grid
- Hover effects for better user interaction
- Dark maroon and baby pink theme

---

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- Git & GitHub

---

## Projects Included

### 1. JavaScript Calculator
A calculator that performs basic arithmetic operations and stores calculation history using arrays and objects.

### 2. Cat Breeds Website
A responsive website using Flexbox to display different cat breeds.

### 3. JavaScript Arrays Project
Manages a store's product inventory using arrays.

### 4. JavaScript Objects Project
Uses objects to manage event attendees efficiently.
